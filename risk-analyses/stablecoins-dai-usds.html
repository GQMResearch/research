DAI / USDS (MakerDAO / Sky)

Overview

  • Type: Crypto-collateralized (CDP) stablecoin
  • Issuer: MakerDAO (now Sky)
  • Backing: Over-collateralized crypto + RWA
  • Market Cap: ~$5B+
  • Chains: Ethereum (native), bridged elsewhere
  • Website: https://makerdao.com / https://sky.money

Rebranding Note

  • MakerDAO → Sky
  • DAI → USDS
  • MKR → SKY
  • Both versions coexist, same backing

Backing & Reserves

Collateral Composition

  • ETH and ETH LSDs (~40%)
  • USDC (PSM - Peg Stability Module)
  • Real World Assets (RWA) - T-bills, loans
  • Other crypto collateral

Collateralization

  • Minimum 150%+ for most vaults
  • System maintains surplus buffer
  • Liquidation mechanisms protect peg

Transparency

  • Fully on-chain and verifiable
  • Real-time monitoring via Makerburn
  • Public governance votes

Issuer Analysis

MakerDAO/Sky

  • Oldest DeFi protocol (2017)
  • Fully decentralized governance
  • DAO-controlled (MKR/SKY holders)
  • No single point of failure

Governance

  • MKR/SKY token voting
  • Delegate system
  • Extensive governance framework
  • Emergency shutdown capability

Historical Performance

Peg Stability

  • Excellent long-term stability
  • Brief deviations during market stress
  • Always recovered
  • ~7 years of track record

Notable Events

  • Black Thursday (March 2020): Some vaults undercollateralized
  • USDC PSM addition improved peg stability
  • No loss of user funds at protocol level

DAI Savings Rate (DSR) / Sky Savings Rate

What is DSR?

  • Yield paid on DAI/USDS deposits
  • Funded by stability fees (borrowing interest)
  • Currently ~5-15% depending on governance

sDAI / sUSDS

  • Yield-bearing wrapped version
  • Earns DSR automatically
  • Used in Pendle, SparkLend, etc.

Risk Factors

Strengths

  • Longest track record in DeFi
  • Fully decentralized governance
  • Over-collateralized design
  • Battle-tested through multiple crises

Risks

  1. USDC Dependency: PSM uses USDC (centralization vector)
  2. RWA Risk: Real world assets introduce counterparty risk
  3. Governance Risk: DAO decisions can change parameters
  4. Complexity: Multiple collateral types add complexity

Governance Attack Vectors

  • Flash loan governance attacks mitigated
  • Time delays on critical changes
  • Emergency shutdown if needed

Summary

DAI/USDS is the most battle-tested decentralized stablecoin. Over-collateralized design, 7+ years of track record, and fully decentralized governance make it a cornerstone of DeFi. Main risks are USDC dependency in the PSM and RWA counterparty risk. DSR provides sustainable yield funded by protocol revenue.
