Arbitrum

Overview

  • Type: Layer 2 (Optimistic Rollup)
  • Parent Chain: Ethereum
  • Launched: August 2021
  • Operator: Offchain Labs
  • Website: https://arbitrum.io
  • Status: Leading L2 by TVL

Security Model

Optimistic Rollup Design

  • Transactions assumed valid unless challenged
  • 7-day challenge period for withdrawals
  • Fraud proofs secure the chain
  • Inherits Ethereum security

Sequencer

  • Currently centralized sequencer (Offchain Labs)
  • Sequencer can censor (temporarily)
  • Cannot steal funds
  • Decentralization roadmap in progress

Fraud Proofs

  • Any validator can challenge invalid state
  • 7-day window for challenges
  • Honest 1-of-N assumption

Trust Assumptions

What You Trust

  1. Ethereum: Base layer security
  2. Fraud Proofs: At least one honest validator
  3. Sequencer (soft): For transaction ordering

What You Don't Need to Trust

  • Sequencer cannot steal funds
  • Can always exit to L1 (worst case)
  • Ethereum settlement guarantees

For DeFi Users

Strengths

  • Low fees (~$0.10-0.50)
  • Fast confirmations (~1 second)
  • Large ecosystem (GMX, Radiant, etc.)
  • $10B+ TVL

Considerations

  • Sequencer centralization
  • 7-day withdrawal delay to L1
  • Bridge risk for non-native assets

Risk Scores

Category Score Notes
Security 4.5/5 Ethereum-backed, fraud proofs

Overall 4.5/5

Summary

Arbitrum is the most established L2 with strong security inherited from Ethereum. Centralized sequencer is the main tradeoff, but it cannot steal funds. Excellent for DeFi with mature ecosystem and low fees.
