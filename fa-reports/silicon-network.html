2. Project Overview

What It Does

Silicon Network tokenizes physical NVIDIA GPUs as NFTs, allowing investors to own fractional or full shares of datacenter hardware earning revenue from AI workloads. Revenue is distributed on-chain to NFT holders in USDC.

3. How It Works (User Perspective)

High-Level Explanation

  1. Datacenter operators deploy NVIDIA GPUs and tokenize them via Silicon Network
  2. Investors buy GPU NFTs (or soon, pool tokens) representing ownership
  3. AI developers rent compute, paying in $SIL tokens → converted to credits
  4. Revenue flows back to NFT holders in USDC based on GPU utilization
  5. Chia blockchain serves as ownership registry with provenance tracking

User Roles & Incentives

GPU NFT Holders

  • Actions: Purchase GPU NFT, hold, collect revenue
  • Incentives: Passive yield from AI compute rentals (10-29% APR claimed)
  • Risks: Depreciation, demand fluctuation, operator performance, illiquidity

Pool Token Holders (Coming Soon)

  • Actions: Buy fractionalized pool tokens for diversified GPU exposure
  • Incentives: Lower minimum investment, diversified across multiple GPUs
  • Risks: Smart contract risk, pool performance variance

Datacenter Operators

  • Actions: Deploy hardware, tokenize via Silicon, manage operations
  • Incentives: Access crowdsourced capital, earn operator fees
  • Risks: Utilization targets, hardware maintenance costs

AI Developers / Compute Buyers

  • Actions: Rent GPU compute via marketplace
  • Incentives: Cheaper than hyperscalers, on-demand access
  • Risks: Availability, performance consistency
