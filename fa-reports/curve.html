2. Project Overview

What It Does

Curve Finance is a decentralized exchange optimized for stablecoin and pegged asset swaps, using a specialized AMM algorithm that minimizes slippage for like-kind assets.

Key Products

  • Stable Pools — Efficient swaps between stablecoins (USDC/USDT/DAI)
  • Crypto Pools — V2 pools for volatile pairs
  • crvUSD — Native overcollateralized stablecoin
  • Lending (LlamaLend) — Soft liquidation lending
