2. Project Overview

What It Does

Merkl is an incentive distribution platform built by Angle Protocol. It enables protocols to efficiently distribute rewards to LPs across multiple DEXs and chains.

Key Features

  • Off-chain computation — Efficient reward calculation
  • Multi-chain — Works across many chains
  • DEX agnostic — Uniswap, Curve, etc.
  • Customizable — Protocols define reward rules

Team

Built by Angle Protocol team (known for ANGLE token and agEUR).
